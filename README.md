# Kota Manohar Portfolio

Personal portfolio for Kota Manohar, a Software Engineer and AI/ML Engineer. It showcases selected work across full-stack engineering, applied AI, data systems, and mobile machine learning.

## Live site

[https://kotamanohar.vercel.app](https://kotamanohar.vercel.app)[![Live Demo](https://img.shields.io/badge/demo-live-brightgreen)](https://kotamanohar.vercel.app)

## Tech stack

- Next.js
- TypeScript
- React
- Tailwind CSS
- Framer Motion
- Redux Toolkit

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Production build

```bash
npm run build
```
